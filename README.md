# 🔢 CALX Scientific Calculator

A modern scientific calculator built using HTML, CSS, and JavaScript.

---

## ✨ Features
- Basic operations (+, −, ×, ÷)
- Scientific functions (sin, cos, tan, log, ln)
- DEG / RAD mode
- Clean and responsive UI
- Toggle for scientific panel
